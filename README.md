# Grau_EET
Coses del grau de EET que necessito tenir al núvol

# Per què aquest repo?
En aquest repositori aniré afegint *carpetes* i *tot el que necessiti* per fer diferents coses, de manera que pugui treballar des de diferents ordinadors, sesne la necessitat d'estar enviant correus amb arxius i tonteries d'aquestes.
Moltes coses les faig directamnet amb el portàtil ja que l'ordinador de casa em comença a fallar una mica, tot i que igualment diria que té força potència igualment.
Segurament falla per tota la merda que hi he descarregat.
En qualsevol cas, aquest serà un espai per poder anar **guardant projectes**, **pràctiques** o el que sigui que vulgui tenir en el núvol.
I escriuré en aquest readme així també **aprenc markdown!**
